# HDF5 Data Viewer & FEBUS Mock Generator

This repository contains tools and a Streamlit application designed to generate, analyze, and visualize HDF5 data files, specifically tailored for simulating and processing Distributed Temperature and Strain Sensing (DTSS) data generated by interrogators like the FEBUS G2-R.

## Features

### 1. HDF5 Data Viewer (`h5_viewer.py`)
A comprehensive Streamlit-based web application to interactively explore HDF5 datasets.
- **Dataset Exploration**: Connect to local `.h5` files and browse the internal structure, groups, attributes, and datasets.
- **Dynamic Slicing**: Visually or numerically slice multidimensional datasets (e.g., Time x Distance). Includes trace averaging functionality across spatial dimensions.
- **Advanced Digital Signal Processing (DSP)**: Apply various digital filters to the data traces directly in the UI, including:
  - Moving Average
  - Savitzky-Golay
  - Median Filter
  - Butterworth Low-Pass / High-Pass
  - Cascading Filter support (sequence multiple filters)
- **Quantitative Analysis**: Set numerical thresholds to automatically detect and list intervals (hotspots or excessive strain anomalies).
- **Spectral Analysis (FFT)**: Generate Fast Fourier Transform plots to analyze spatial frequencies.
- **Export capabilities**: Download the sliced and filtered data as CSV files for external use.

### 2. Mock Data Generators (`generate_mock_febus_*.py`)
Scripts to generate synthetic HDF5 datasets with injected thermal and strain anomalies to test analysis systems.
- Configurable base temperatures and strains with controlled signal-to-noise ratios.
- Pre-programmed hotspots and strain events (e.g., solar heating, leaks, landslides) spanning multiple structural indices.
- Exports standard `.h5` arrays and metadata attributes compatible with the viewer application.

### 3. HDF5 Utilities 
Various python scratch scripts demonstrating how to interact with the `h5py` library for listing structures, extracting metadata, and slicing datasets.

## Installation

Ensure you have Python 3.8+ installed.

1. Clone this repository.
2. Install the required dependencies:
   ```bash
   pip install numpy pandas matplotlib scipy h5py streamlit
   ```

## Usage

**To run the Data Viewer App:**
```bash
streamlit run h5_viewer.py
```

**To generate a 10km Mock Dataset:**
```bash
python generate_mock_febus_10k.py
```
